New Features
=============
* Support for select OpenCL 1.2 APIs including:
    * clCompileProgram()
    * clLinkProgram()
* Support for OpenCL 1.2 clGetProgramBuildInfo query
    * CL_PROGRAM_BINARY_TYPE

* `AM572 EVM`_ (Linux and TI-RTOS hosts)
* `AM571 EVM`_
